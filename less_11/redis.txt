1. В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.

Будем использовать для этого тип данныйх zset (Sorted Set)

1. Создадим множесто: SADD incoming_ip:count 0 1.1.1.1
2. Для каждого подключения, боавляем новый ip или изменяем рейтинг уже существующего в множестве:  ZINCRBY incoming_ip:count 1 192.168.1.4
3. Посмотреть список IP  с количеством посещений: ZRANGE incoming_ip:count 0 -1 WITHSCORES
4. Узнать количесвто посещений с IP: ZSCORE incoming_ip:count 192.168.1.4
